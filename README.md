# Health-Sync-Web-App
Health Sync – Smart Health Monitoring Web App – A web system where doctors can monitor patients’ real-time health data (from smartwatches), manage records, and communicate through an interactive dashboard. Includes authentication, Firebase database, and responsive UI design.
